# “Automotive Data Analysis in Excel: Pricing, Fuel Efficiency, and Torque Insights”

**🚗 Automotive Data Analysis (Excel Project)**

**📊 Overview**
This project showcases an Excel-based data analysis of various car models, focusing on pricing, fuel efficiency, and performance metrics such as torque. It demonstrates the use of formulas, calculated fields, and pivot tables to extract meaningful insights from raw automotive data.

**📁 File Structure**

**1. Data Sheet**
Contains the raw dataset of vehicles
Key fields include:
- Make & Model
- Invoice Price
- MSRP
- Fuel Economy
- Torque
-   Additional calculated column:  
   - Price per Average Fuel Economy

**2. Completed Sheet**
Cleaned and structured dataset
Includes:
Derived metrics (e.g., Price per Average FE)
Organized fields for easier analysis
Used as the base for pivot tables

**3. Price Analysis**
Focuses on:
Invoice price per car model
Helps identify:
Price distribution across brands
High vs. low-cost vehicles

**4. Torque Analysis**
Highlights:
Torque values per vehicle
Comparison of performance across models
Useful for:
Performance benchmarking

**📌 Key Formulas Used**
Price per Average Fuel Economy
= Invoice Price / Average Fuel Economy
Data cleaning and referencing formulas
Lookup and structured referencing (where applicable)

**📈 Key Insights**
Premium brands (e.g., Audi, BMW) show higher price-to-efficiency ratios
Electric vehicles tend to have:
High torque at low RPM
Higher pricing but better performance metrics
Significant variation exists between models within the same brand
